# Dane źródłowe

Katalog zawiera pliki pobrane ze źródła bez zmian wynikających z modelu analitycznego.

## Pliki

| Plik | Zawartość | Rekordy | Kolumny |
| --- | --- | ---: | ---: |
| `DataCoSupplyChainDataset.csv` | Zamówienia, klienci, produkty, sprzedaż i wysyłki | 180 519 | 53 |
| `DescriptionDataCoSupplyChain.csv` | Opis pól pliku głównego | 52 | 2 |

Plik główny wykorzystuje kodowanie ISO-8859. Przy imporcie do Power Query lub innego narzędzia należy sprawdzić kodowanie przed zmianą typów danych.

## Źródło

## Zasady wykorzystania

- Pliki w tym katalogu służą jako materiał wejściowy.
- Czyszczenie i modelowanie odbywa się poza katalogiem `raw`.
- W raporcie nie są wykorzystywane imiona, nazwiska, adresy e-mail, hasła ani dokładne adresy klientów.
- Przed końcową publikacją repozytorium należy rozważyć usunięcie surowego pliku z GitHub i pozostawienie instrukcji pobrania danych ze źródła.
